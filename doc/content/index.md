        MASTODON is built on the
        <a href="http://mooseframework.org/">
          MOOSE framework
        </a>
        and can efficiently solve problems using standard workstations or very
        large high-performance computers in different dimensions,
        including full 3D, 2D, 2D-RZ axisymmetric, layered 1D, and spherical 1D systems.
        Code reliability is paramount for the MOOSE framework, and MOOSE-based applications.
        MASTODON therefore employs a well-defined development and testing strategy in which,
        code changes are only merged into the repository after both a manual
        code review and an automated regression test have been completed.
        </a>
      </p>
